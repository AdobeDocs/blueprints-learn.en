# Improve customer engagement

Increase interaction frequency and depth across all digital and physical touchpoints. This objective focuses on delivering relevant, timely interactions that encourage deeper relationships and more frequent engagement with the brand across channels.

## Key performance indicators

The following KPIs measure progress toward this objective.

| KPI | Description |
| --- | --- |
| Engagement | Composite measure of interaction frequency, depth, and breadth across channels |
| Time On Page | Average time visitors spend engaging with web content |
| Open Rates | Email and message open rates indicating engagement with outbound communications |
